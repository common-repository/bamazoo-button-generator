A plugin to enable easy integration of bamazoo buy now and add to cart buttons into a WordPress post or page with [dgs] shortcodes.

== Description ==

Sell downloads on your WordPress powered site with ease! Install this plugin to enable easy integration of the bamazoo service https://bamazoo.com buy now and add to cart buttons into a WordPress post or page with [dgs] shortcodes.

For a graphical overview of the installation and usage visit:- https://bamazoo.com/support/using-the-wordpress-plugin

== Installation ==

1. Upload `bamazoo-Button-Generator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste the [dgs] shortcode generated by the bamazoo button wizard into WordPress posts or pages where you want your buy now or add to cart buttons to appear.
